Adds the ability to view your posts and pages at different viewport sizes with a click of a button

== Description ==

This plugin provides you an easy quick way to test your content looks good at various sizes.

Features of the plugin include:

* Mobile view
* Tablet view

**Getting setup with the plugin**

[youtube http://www.youtube.com/watch?v=bXe5_BvvtxA]

[Plugin Support & Feature requests](http://web-design-weekly.com/support/forums/forum/responsive-post-preview-plugin/).

== Installation ==

1. Activate the plugin

2. View the edit post screen to start using it.

== Frequently Asked Questions ==

=How does it work?=

Once activated two icons will be visible under the "Preview" post button. These icons open up the post in a new tab at a specific view-port size.

[youtube http://www.youtube.com/watch?v=bXe5_BvvtxA]
