== Description ==

Adds the ability to view your posts at different viewport sizes with a click of a button

== Installation ==

1. Activate the plugin

== Frequently Asked Questions ==

=How does it work?=

Once activated two icons will be visible under the "Preview" post button. These icons open up the post in a new tab at a specific view-port size.
